# How to run code

## Clone the repository

```sh
git clone https://github.com/Kamanashis-Biswas/Assessment-SeoPage1.git
```

Now, open the folder in your <b>IDE (VS Code recommended)</b> and run the following commands on <b>terminal (Git Bash)</b>

```bash
cd backend
```

```bash
npm install
```

```bash
npm start
```
